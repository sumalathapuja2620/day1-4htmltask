HTML

INTRODUCTION:

HTMLwas created by Tim bernes Lee and organized by W3C(World Wide Web Consortium).

WHAT IS HTML?

HTML, or Hypertext Markup Language, is the code used to structure and define the content of a web page. To create a only static web pages.

*Hypertext:Collection of text and links.

*Mark-up:Gives instructions to the browser like how to display the data.

*Language:To interact with the browser.

Keywords:

Pre-defined logic written by language itself.

what is syntax:

syntax to ensure that a computer can read and interpret the code.

TAGS:

WHAT IS TAG?

The keywords which is enclosed between to display the data in browser, tags. Every tag has its own behaviour.

TYPES OF TAGS:

1.Paired tag (or) Container tag:The paired tag which is having opening and closing tag is known as paired tag or container tag. paired tags have p,h1-----h6,div,span,table.

2.unpaired tag (or) Unpaired tag:The tag which closes itself is having closing tag or unpaired tag. Unpaired tags have img,br,hr,input.

STRUCTURE OF HTML:

!Doctype html(Defines latest version of html.)

root element(Parent of all tags.)

head element(Gives information about our document.)

body element(Display the content on the browser)

ELEMENTS:

WHAT IS ELEMENT?

The combination of opening, closing and width content between the tag is known as elements.

TYPES OF ELEMENTS:

1.Block level element:The tag which occupies whole width and creates a new line.

EX:h1---h6,p,div,semantic elements.

2.Inline-Block level element:The elements which occupies only required space and does not creats a new line and we can apply width,height,margin, and padding properties.

Ex:img,iframes,input,button.

3.Inline element:The elements which occupies only required space and does not create a new line, in CSS we can not apply width, height, padding top and bottom and margin top and bottom.

EX:span,formatting tags,b,strong,i.
